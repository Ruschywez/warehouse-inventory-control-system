<h1 align="center">Склад информационная система</h1>
Тренировочный проект по созданию системы отвечающей за учет товаров на складе.

## Установка зависимостей в окружение

```cmd pip install -r requirements.txt```

## Пример с запуском сервера
```cmd uvicorn main:app --reload --host 0.0.0.0 --port 8000```

## Настрйока подключения к базе данных
Сервер написан на основе PostgreSQL и требует наличия готовой базы данных и пользователя.

#### База должа содержать следующие таблицы:
    - goods
    - remainder
    - receipt
    - shipment
Подробнее о содержании базы данных смотрите в документации.
#### Пользователь должен иметь следующие права:
    - SELECT
    - INSERT
    - UPDATE
    - DELETE
